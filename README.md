To compile use  ```make```

To check that everything is installed correctly type : 

```shell
./bin/main/Segmentate img/fille.ppm output.ppm 64 64 
```

To segmentate use : 
```shell
./bin/main/Segmentate <filename> <filename_output> <Width_nb_of_block> <Height_nb_of_block> [<threshold> : 1.01]
```
